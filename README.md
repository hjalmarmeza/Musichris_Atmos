# 🌌 MusiChris Atmos Engine V6.2

**El corazón visual y espiritual de MusiChris Studio.**

Atmos es un motor de producción ministerial automatizado diseñado para generar contenido cinematográfico de alta fidelidad, fusionando paisajes majestuosos con música de paz y reflexiones que alimentan el alma.

## 🚀 Capacidades Actuales
- [x] **Motor V6.2 Dinámico:** Generación de videos de 1 hora con 15+ temas musicales.
- [x] **Reflexiones Espirituales:** Sistema de "Soul-Feeding Phrases" distribuido aleatoriamente.
- [x] **Estética Premium:** Cierres con logo animado transparente y "Glassmorphism" visual.
- [x] **Control Remoto:** Gestionable desde GitHub / Celular.

## 🛠️ Tecnologías
- **Core:** Python 3.10+
- **Motor de Video:** FFmpeg 8.0 (Optimizado con `colorkey` y `filter_complex`)
- **Diseño Dinámico:** Pillow (PIL) para capas tipográficas en *Baskerville*.
- **Dashboard:** Vanilla JS / HTML5 con estética Dark-Glass.

## 📂 Estructura del Proyecto
- `/scripts/atmos_engine.py`: El cerebro de renderizado.
- `/data/musichris_master_catalog.json`: El catálogo musical sincronizado.
- `/ui/`: Panel de control interactivo.
